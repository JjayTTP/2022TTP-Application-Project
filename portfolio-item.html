Tip Calculator

IOS Application

As part of the IOS development course at Code Path, I had to make a Tip Calculator App.The user can enter a bill amount, choose a tip percentage, and see the tip and total values. The user can select between tip percentages by tapping different values on the segmented control and the tip value is updated accordingly.
