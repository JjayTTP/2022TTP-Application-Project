Flixster Movies

IOS Application

As part of the IOS development course at Code Path, I had to create a Flixster Movie Application. This application is a movie browsing app, similar to Fandango and Rotten Tomatoes, that lets a users browse through a list of movies.

The user is able to see an app icon on the home screen and a styled launch screen. The user can view and scroll through a list of movies now playing in theaters. The user can view the movie poster image for each movie.

The user can tap a cell to see more details about a particular movie. The user can tap a tab bar button to view a grid layout of Movie Posters using a CollectionView.
