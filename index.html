Portfolio

A look into some projects I've built

Movie Library
Movie Library is an IOS application where a user can store any movie they want to watch in there app. Also, the user will have the options to look at what current movies are showing on theatres, and will have access to any reviews people may have written about the movie, and will see which actors were a part of the film.
Tweet
Tweet is a an Twitter clone app, that I built through CodePath. In this app the user is able to sign in/out of Twitter, like tweets, see tweets of people they follow, and post a tweet. Lastly, the user is able to also refresh for new tweets and scroll indefinitely to load past tweets.
Guess the Number
In this project, I used HTML/CSS/Javascript to build a Guessing game. The user will be able to guess a number from 1-100. If they get the correct number, the screen will turn green and their updated highscord will be 10 now. Each user has 10 tries to get the correct number. Once the score reaches 0, the user loses.
